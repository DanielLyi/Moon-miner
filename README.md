# Tutorial for installing Game

1) Download .zip file from [release](https://github.com/DanielLyi/Moon-miner/releases/tag/0.0.1)
2) Unarchive it in any folder  
3) Click twice on Moon Miner.exe
4) Enjoy the game ^^
